# 🛵 Colmado Dash

> Dominican Republic retro arcade courier game — built with SpriteKit

**You are a courier in Santo Domingo, DR.** Pick up deliveries from colmados (corner stores), dodge saboteurs, throw plátanos at enemies, and deliver the goods!

## 🎮 Features

- 🏙️ **Top-down Santo Domingo streets** — colorful buildings, palm trees, roads
- 🛵 **4 vehicles** — Bicicleta, Moto, Carro, Concho Taxi
- 💣 **4 weapons** — Plátano (boomerang), Huevo (slime), Salami (grenade), Peo (freeze cloud)
- 😈 **3 enemy types** — Saboteurs, Swervy cars, Police
- 🍺 **Power-ups** — Presidente beer, Mangú, Café
- 💵 **Economy** — Earn cash, buy upgrades, unlock vehicles
- 📋 **5 missions** — Escalating difficulty across Santo Domingo neighborhoods
- 🕹️ **Virtual joystick** — Smooth touch controls
- 🎨 **Zero image assets** — All sprites drawn programmatically
